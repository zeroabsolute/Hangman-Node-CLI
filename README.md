## HANGMAN GAME
*Node.js CLI program that simulates Hangman Game*

**Requirements:**
Node.js >= v.10

**Run the program:**

    git clone https://github.com/ghaxhi/Hangman-Node-CLI.git Hangman
    cd Hangman/src
    node index.js
